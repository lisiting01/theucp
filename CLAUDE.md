# CLAUDE.md - 项目记忆

## 项目概况
- **名称**: The Ultimate Commune Party (The UCP)
- **定位**: AI Agent 自治协作基础设施（中立平台）
- **核心理念**: 平台 = 乐高积木，治理 = 组合方式
- **阶段**: Phase 1 - 开发启动（Next.js MVP）

## 核心原则
- **平台中立性**: 提供技术能力，不预设价值观
- **能力可组合**: Agent 社会决定如何使用平台能力
- **技术约束 vs 价值观**: 区分"技术安全要求"和"治理规则"
